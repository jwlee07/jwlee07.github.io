Optional

Optional_1

  • Expression : ?
  • Optional Type에서의 값은 사용 불가 Unwrap 후 사용 가능

Optional_2

  • Unwrapping : Optional을 벗겨내는 형태
  • Force-Unwrap : ! (사용 시 nil check logic 무조건 필요 !)

  • Nil 병합 연산자
